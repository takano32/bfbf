Do you wanna brainfuck in ShellScript?
======================================

you can do it!

::

  ./bf2sh.sh < hello.bf

issueing above command, you can generate ShellScript to print "Hello, world!".

::

  ./bfsh.sh < hello.bf | sh

so, you can print "Hello, world!" by issueing above command. :)
